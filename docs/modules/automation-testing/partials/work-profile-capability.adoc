Launch a Work Profile app using the name shown in the app drawer (case-sensitive). Only 1 Work Profile app can be launched in a session.

* *Type*: `string`
* *Required capabilities*: None
* *Optional capabilities*: None

.Example
[source,java]
capabilities.setCapability("kobiton:workApp", "Firefox");

[NOTE]
====

* This capability cannot be used in Basic Appium 2 sessions.

* The following capabilities will be ignored when using `kobiton:workApp`:

** `app`

** `appium:appPackage`

** `appium:appActivity`

====
