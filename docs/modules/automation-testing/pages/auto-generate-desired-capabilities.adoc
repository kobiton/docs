= Auto-generate desired capabilities
:navtitle: Auto-generate desired capabilities

Learn how to auto-generate desired capabilities for one of your test devices. When you're finished, you can xref:modify-desired-capabilities.adoc[add them to your test suite].

[NOTE]
xref:list-of-desired-capabilities.adoc#_manually_added_parameters[Some desired capabilities], like `flexCorrect` and `visualValidation`, will need to be added manually.

== Open the automation settings

Once you've xref:devices:find-a-device.adoc[found a device], select the *vertical ellipses* to open the device overview.

image:open-device-overview-context.png[width=1400, alt=""]

Next, select the *Automation Settings*.

image:device-overview-closeup.png[width=750, alt=""]

== Choose capabilities

Select a capability from each dropdown to start auto-generated your `DesiredCapabilities` object. You can learn about each of our capabilities by xref:list-of-desired-capabilities.adoc[referencing the list].

image:auto-generate-desired-capabilities-context.png[width=1200, alt=""]

Here's an example:

[source,java]
----
 // Auto-generated capabilities

String kobitonServerUrl = "https://<user-name>:<api-key>@api.kobiton.com/wd/hub"; // <user-name> and <api-key> will be automatically replaced with your credentials.

desiredCapabilities.put("sessionName", "Biometric authentication");
desiredCapabilities.put("sessionDescription", "Testing biometric authentication for the login screen.");
desiredCapabilities.put("deviceOrientation", "portrait");
desiredCapabilities.put("noReset", true);
desiredCapabilities.put("fullReset", false);
desiredCapabilities.put("captureScreenshots", true);
desiredCapabilities.put("useConfiguration", "");
desiredCapabilities.put("autoWebview", true);
desiredCapabilities.put("browserName", "safari");
desiredCapabilities.put("groupId", 0011);
desiredCapabilities.put("deviceGroup", "ORGANIZATION");
desiredCapabilities.put("deviceName", "iPhone XS");
desiredCapabilities.put("tagName", "");
desiredCapabilities.put("platformName", "iOS");
desiredCapabilities.put("platformVersion", "15.5");
----

When you're finished, select the *Copy to clipboard* icon and xref:modify-desired-capabilities.adoc[add the capabilities to your test suite].

image:auto-generate-desired-capabilities-closeup.png[width=750, alt=""]
