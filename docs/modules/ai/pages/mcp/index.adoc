= MCP
:navtitle: MCP

Kobiton uses Model Context Protocol (MCP) to connect AI coding assistants such as Claude, Copilot, and Gemini to the Kobiton platform. This integration lets you manage devices, upload apps, run tests, and review results without switching between tools or navigating the Kobiton Portal.

Common tasks include:

- Check device availability
- Reserve devices for testing
- Upload app builds
- Run automation or scriptless tests
- Review session results and artifacts

== Before you begin

To use Kobiton with MCP, you need:

- A Kobiton account
- A supported AI assistant installed locally
- Access to the Kobiton plugin or connector
