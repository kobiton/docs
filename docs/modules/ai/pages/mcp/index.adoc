= MCP
:navtitle: MCP

Use Kobiton with Claude Code to bring AI-assisted mobile testing into your terminal. With the Kobiton plugin for Claude Code, you can manage devices, upload apps, run tests, and review results by asking Claude what you want to do.

Kobiton connects Claude Code to the Kobiton platform through an MCP server and guided skills. This helps you complete common mobile testing tasks without switching between tools or manually navigating the Kobiton portal.

Common tasks include:

* Checking device availability
* Reserving devices for testing
* Uploading app builds
* Running automation or scriptless tests
* Reviewing session results and artifacts

== Before you begin

To use Kobiton with Claude Code, you need:

* A Kobiton account
* Claude Code installed locally
* Access to the Kobiton plugin and MCP connection

For setup and authentication steps, see xref:pages/mcp/claude-code.adoc[Get started with Kobiton for Claude Code].
