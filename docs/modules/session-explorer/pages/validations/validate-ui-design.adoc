= Validate the UI design
:navtitle: Validate the UI design

Learn how to validate your app or website's UI design so you can find and fix inconsistencies between the original design and the final product.

== Before you start

You'll need to xref:integrations:figma/figma.adoc[connect Figma to your Kobiton account].

== Open UI validation

First, search for a session and xref:session-explorer:launch-session-explorer.adoc[launch Session Explorer].

image:$NEW-IMAGE$[width=, alt=""]

In Session Explorer, select *Compare to UI Design*.

image:$NEW-IMAGE$[width=, alt=""]

== Choose a Figma project

Choose the Figma project to use to compare UI design. All frames from the project will be listed below.

image:session-explorer:figma-project.png[width=1000, alt=""]

== Change the resolution

Choose a resolution from the Figma project. By default, *All Resolutions* is selected.

image:session-explorer:all-resolutions.png[width=500, alt=""]

== Search for a frame

Filter the list of frames by a specific frame name.

image:session-explorer:search-frame.png[width=500, alt=""]

== Select a frame

All frames from the selected project.

image:session-explorer:frames-list.png[width=500, alt=""]

== Upload an image to compare

Upload an image to use to compare UI design.

image:session-explorer:upload-screen.png[width=500, alt=""]
