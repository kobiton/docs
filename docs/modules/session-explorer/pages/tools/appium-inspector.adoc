= Appium inspector
:navtitle: Appium inspector

The Appium inspector in the Session Explorer lets you copy xpath values, inspect UI elements, download your inventory, and more.

image:$NEW-IMAGE$[width=, alt=""]

== Refresh inventory

Display the XML hierarchy of the current device screen. Select a specific element from the hierarchy to highlight it on the device.

image:$NEW-IMAGE$[width=, alt=""]

image:refresh-inventory-closeup.png[width=500,alt="A closeup to see Refresh Inventory"]

[NOTE]
*Refresh inventory* resets all filters and searches.

== Inspect elements

Hover over on-screen elements to highlight the elements in the XML hierarchy.

image:inspect-elements-context.png[width=1000,alt="A context to Inspect Element function in Appium Inspector"]

== Download inventory

Download the current XML hierarchy as an `.xml` file.

image:session-explorer:download-inventory-context.png[width=1000,alt="A context to Download inventory of Appium Inspector"]

== WebView

Filter the XML hierarchy by the selected WebView. The default WebView is `NATIVE_APP`.

image:webview-closeup.png[width=500,alt="A closeup to webview with default is NATIVE_APP"]

== Search bar

Filter the XML hierarchy by a specific element.

image:search-bar-closeup.png[width=500,alt="A closeup to search bar in the Appium Inspector"]

== Attributes

Display an element's attributes.

image:attributes-context.png[width=1000,alt="A context to Attributes of Appium Inspector"]

Hover over a specific attribute and to copy it to your clipboard.

image:hover-attribute-closeup.png[width=500,alt="Hover over a specific attribute and to copy it to your clipboard"]
