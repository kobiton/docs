= Rewatch a test session
:navtitle: Rewatch a test session

Learn how to rewatch a test session so you can review and share specific test steps from the session.

== Open the session timeline

First, search for a session and xref:session-explorer:launch-session-explorer.adoc[launch Session Explorer].

image:$NEW-IMAGE$[width=, alt=""]

== Watch the session video

Play or pause the session recording.

image:session-explorer:play_pause-closeup.png[width=400,alt="A closeup to play/pause button in the timeline"]

Zoom in or out of the session timeline.

image:session-explorer:zoom.png[width=1000,alt="A closeup to Zoom function on the timeline"]

== Review a test step

Display the selected test step on the device screen.

image:session-explorer:test-steps-closeup.png[width=1000,alt="A closeup to test steps displayed on the timeline"]

The related action is marked in red.

image:session-explorer:red-dot-context.png[width=1000,alt="The related action is marked in red."]

Skip to the previous or next test step.

image:session-explorer:previous-next-closeup.png[width=1000,alt="A closeup to Previous & Next button on the timeline"]

== Get a test step's ID

Copy the unique id assigned to the current test step.

image:session-explorer:copy-command-id-closeup.png[width=1000,alt="A closeup to copy command ID function on the timeline"]

== Share a test step

Copy a link to the current test step and share it with others easily.

image:session-explorer:copy-link-closeup.png[width=1000,alt="A closeup to copy link function on the timeline"]

== Download a screenshot

Download a screenshot of the selected test step.

image:session-explorer:download-screenshot-closeup.png[width=500,alt="Download button to download screenshot of the selected step"]
