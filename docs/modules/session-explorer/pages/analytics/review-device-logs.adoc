= Review device logs
:navtitle: Review device logs

Learn how to review device logs after your test session so you can search for specific log messages and share them with your team.

== Open the device logs

First, search for a session and xref:session-explorer:launch-session-explorer.adoc[launch Session Explorer].

image:launch-session-explorer-context.png[width=1000,alt="A context to Session Explorer launched"]

In Session Explorer, select *Review Device Logs*.

image:session-explorer:review-device-logs-context.png[width=1000,alt="A context to Review Device Logs feature in Session Explorer"]

== Filter the logs

Filter logs to only display *Notice*, *Info*, *Warning*, or *Error* messages. By default, the filter is set to *All*.

image:session-explorer:review-device-logs-filter-closeup.png[width=500,alt="A closeup to the filter of Review Device Log feature"]

== Search for a message

Filter the logs by specific criteria.

image:session-explorer:review-device-logs-search-closeup.png[width=500,alt="A closeup to the filter of Review Device Log feature"]

== Copy log messages

Copy the current logs to your clipboard.

image:session-explorer:review-device-log-copied-context.png[width=1000,alt="A current log is copied in Review Device Log"]
