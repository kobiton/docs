// Retrieve API base URL

The API base URL is used to make requests to Kobiton API. To retrieve the URL:

. Log in to the Kobiton Portal using your Kobiton username/email and password.

. Select the *Help* icon, then choose *API*.

+

image:help-widget:help-widget.png[width=400,alt="A closeup to the help widget with API option"]

. You will be taken to the API documentation home page. Note down the value of *Base URL (Rest)*.

+

image:help-widget:api-home-page.png[width=1000,alt="The API documentation home page with Base URL visible"]
