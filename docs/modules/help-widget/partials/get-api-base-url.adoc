// Retrieve API base URL

Use the API base URL to make requests to the Kobiton API. To retrieve the URL:

. Sign in to the Kobiton Portal.

. Select the *Help* icon, then choose *API*.

+

image:help-widget:help-widget.png[width=400,alt="Help widget open in the Kobiton Portal"]

. The API documentation home page opens. Copy the *Base URL (Rest)* value.

+

image:help-widget:api-home-page.png[width=1000,alt="The API documentation home page with Base URL visible"]
