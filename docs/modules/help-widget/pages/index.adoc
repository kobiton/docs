= Help widget
:navtitle: Help widget

Open the help widget to access documentation and reference resources from the Kobiton Portal.

== Launch and use the widget

Open the help widget from anywhere in the Kobiton Portal by selecting the *Help* icon in the navigation bar.

image:help-widget:help-icon.png[width=100,alt="A closeup to the help icon"]

image:help-widget:help-widget.png[width=400,alt="Help widget open in the Kobiton Portal"]

While the help widget is open, you can:

* Search for documentation using the search bar
* Access the documentation (*Help*), API reference (*API*), or *Support* site
* Check the current Portal version
* Resize the widget by dragging the bottom right corner
* Close the widget by selecting the *X* icon
