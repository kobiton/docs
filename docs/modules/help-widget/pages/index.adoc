= Help widget
:navtitle: Help widget

Use the help widget to quickly access documentation and reference resources from the Kobiton Portal.

== Launch and use the widget

The widget can be launched anywhere in the Kobiton Portal by selecting the *Question mark* icon in the navigation bar.

image:help-widget:help-icon.png[width=100,alt="A closeup to the help icon"]

While the help widget is open, you can:

image:help-widget:help-widget.png[width=400,alt="A closeup to the help widget with all features visible"]

* Search for documentation using the search bar
* Access the documentation (Help), API reference, and support site
* Check the current Portal version
* Resize the widget by dragging the bottom right corner
* Close the widget by selecting the *X* icon
