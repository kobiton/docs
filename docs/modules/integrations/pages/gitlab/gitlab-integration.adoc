= GitLab integration
:navtitle: GitLab

Use GitLab CI/CD pipelines with Kobiton mobile testing workflows.

The GitLab integration repository includes sample pipelines for:

* Uploading apps to Kobiton
* Running automation tests on Kobiton devices

For setup instructions and sample configurations, see the
link:https://github.com/kobiton/GitLab[Kobiton GitLab integration repository,window=read-later].

== Repository contents

[cols="1,3", options="header"]
|===
| Directory | Description

| `samples/upload-app`
| Sample GitLab pipeline for uploading an app to Kobiton.

| `samples/automation-test-nodejs`
| Sample Node.js automation test pipeline.

| `samples/execute-test`
| Sample workflow for executing automation tests on Kobiton.
|===
