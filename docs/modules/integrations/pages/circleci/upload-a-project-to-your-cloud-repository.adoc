= Upload a project to your cloud repository
:navtitle: Upload a project to your cloud repository

Learn how to upload your project files to a cloud repository (like link:https://github.com/about[GitHub] or link:https://about.gitlab.com/[GitLab]), so you can xref:integrations:circleci/run-an-automation-test.adoc[run an automation test using CircleCI].

[NOTE]
If your project is already hosted in a cloud repository, skip to xref:integrations:circleci/run-an-automation-test.adoc[].

You'll need to set up SSH for your cloud repository account.
