== Manage your credentials

[#_get_server_url]
=== Get the Appium server URL

The Appium server URL is used for Appium automation testing. Select the *Copy* icon to add it to your clipboard.

image:profile:server-url-closeup.png[width=500,alt="A Server URL section in API Keys"]

[#_get_an_api_key]
=== Get an API key

A default API key is automatically generated when you create an account. Select the *Copy* icon to add it to your clipboard.

image:profile:copy-api-closeup.png[width=500,alt="Click the Copy icon to copy API"]

=== Create a new API key

To create a new API key, select *Add key*.

image:profile:add-key-closeup.png[width=500,alt="A closeup to select Add key in API keys"]

=== Rename an API key

To rename an API key, select the key's current name, then enter a new one.

image:profile:renamed-api-closeup.png[width=500,alt=""]
