    = Manage your API keys
:navtitle: Manage your API keys

Learn how to manage your API keys and server URL so you can add them to your xref:automation-testing:scripting/auto-generate-an-appium-script.adoc[Appium scripts] and xref:integrations:index.adoc[integration] configuration files. In your xref:profile:view-your-profile.adoc[profile settings], select *API Keys* to get started.

image:profile:api-key-context.png[width=1000,alt="Select API Keys on the top nav bar"]

== Get the server URL

A server URL is automatically generated when you create an account. Select the *Copy* icon to add it to your clipboard.

image:profile:server-url-closeup.png[width=500,alt="A Server URL section in API Keys"]

== Get an API key

A default API key is automatically generated when you create an account. Select the *Copy* icon to add it to your clipboard.

image:$NEW-IMAGE$[width=, alt=""]

== Create a new API key

To create a new API key, select *Add key*.

image:$NEW-IMAGE$[width=, alt=""]

== Rename an API key

To rename an API key, select the key's current name, then enter a new one.

image:$NEW-IMAGE$[width=, alt=""]
