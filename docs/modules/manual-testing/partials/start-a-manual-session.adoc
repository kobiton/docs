== Choose a device

In Kobiton, select *Devices*.

image:select-devices-context.png[width=1000,alt="Select Devices tab"]

Use views, categories, statuses, and xref:devices:search-for-a-device.adoc[more] to search for a device.

image:manual-testing:search-for-a-device-context.png[width=1000,alt="Choose a device to launch a manual session"]

== Start a session

Once you've found a device, select *Launch* to start a manual session.

image:manual-testing:launch-a-device-closeup.png[width=500,alt="A closeup to select Launch to start a manual session"]
