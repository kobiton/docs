= Enter an Apple ID verification code during a manual session

During a manual iOS session, Apple might block direct text entry for protected Apple ID verification fields. When this happens, use your physical keyboard to enter the verification code or Mac password shown on the device screen.

== Complete Apple ID verification

. On the device screen, open Settings and start the Apple ID sign-in process.
. Follow the prompts on the device screen.
. If the Apple ID Verification Code prompt appears, type the six-digit verification code on your physical keyboard.
+
The verification code appears in a separate window on the Mac mini host. After you enter the code, the verification window closes and the Apple ID sign-in process continues.
. If the Enter Mac Password prompt appears, type the password for the Mac mini host on your physical keyboard.
. If the Apple Security Enablement window appears in Portal, do not type in that window. Continue typing on your physical keyboard to enter the value on the device screen.
+
image:manual-testing:enter-mac-pw.png[width=400, alt="The Apple Security Enablement window appears in a separate window"]
+
. If the iCloud merge prompt appears for Safari data and Freeform, select **Don’t Merge**.

When the Apple ID name and email appear on the device screen, sign-in is complete.
