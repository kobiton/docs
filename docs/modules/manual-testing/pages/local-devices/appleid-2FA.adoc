= Sign in to an Apple ID during a manual session

== Sign in to the Apple ID

. On the device screen, start the Apple ID sign-in and continue until the *Apple ID Verification Code* prompt appears with the message "A message with a verify code has been sent to your other devices."
. Locate the six-digit verification code. Kobiton displays the code in a new window on the Mac mini host.
. Click the verification code field on the device screen to focus it, then type the code on your physical keyboard.
. When the *Enter Mac password* prompt appears on the device screen, note the host ID shown. The host ID resembles a serial number, not an IP address. The prompt asks for the Mac password, not the iCloud password.
. When the *Apple Security Enablement* window opens in Portal, follow the instruction shown: "Type on your physical keyboard to enter this Apple-protected field." Do not type into the Portal window. Leave the *Input as sensitive data* toggle at its default; it does not affect this workflow.
. Type the Mac password on your physical keyboard. The value populates the *Enter Mac password* field on the device screen.
. Submit the password to continue.
. On the device screen, when the iCloud merge prompt appears for Safari data and Freeform, select *Don't Merge*.

== Verification

The Apple ID name and email appear on the device screen. Sign-in is complete.
