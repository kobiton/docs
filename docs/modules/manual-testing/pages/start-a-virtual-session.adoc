= Start a virtual session
:navtitle: Start a virtual session

Start a manual test session on a virtual device in Kobiton Portal. The session behaves the same as a session on a real device, including the xref:manual-testing:device-controls.adoc[device controls].

== Prerequisites

Your organization's admin must enable virtual devices. Contact your admin if the *virtual devices* tab does not appear in the device list.

== Start the session

. Sign in to Kobiton Portal.
. In the device list, select the *virtual devices* tab.
+
// Screenshot needed: Kobiton Portal device list with the virtual devices tab visible. Source: Tiffany to capture from portal.kobiton.com.
image:manual-testing:virtual-devices-tab-context.png[width=1000,alt="The virtual devices tab in the device list"]
. Find a virtual device, then select *Launch* to start the session.

== Find a virtual session in the session list

After the session ends, search for `is:virtual` on the Sessions page to filter the list to virtual-device sessions.
