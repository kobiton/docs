= Change a user's team role
:navtitle: Change a user's team role

Learn how to change a user's team role, so they can be made a *Team Owner*, *Team Admin*, or *Team Member*.

[NOTE]
Learn more about xref:organization:teams/team-roles-and-permissions.adoc[team roles].

== Change a user's team role

In *Org Management*, select *Teams*, then choose your team.

image:organization:select-a-team-context.png[width=1000,alt="Select a team in Teams tab"]

Use the search bar to find the user, then select the *minus* icon to remove the user.

image:organization:change-user-team-role-context.png[width=1000,alt="Use the search bar to find the user, then select the minus icon to remove the user"]

Next, select *Add new user*, then choose a role: *Team Owner*, *Team Admin*, or *Team Member*.

image:organization:add-team-role-closeup.png[width=500,alt="Add a Team Role to a user"]

After you choose a role, search for the user you removed previously.

image:organization:change-users-team-role-closeup.png[width=500,alt="A closeup to change users team role"]

Choose the user, then select *Update*.

image:organization:select-update-closeup.png[width=500,alt="Select Update after changing the user's team role"]

When you're finished, select *Save*.

image:organization:click-save-context.png[width=1000,alt="Click Save after changing the user's team role"]
