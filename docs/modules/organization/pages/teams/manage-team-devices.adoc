= Manage team devices
:navtitle: Manage team devices

Learn how to manage your team devices, so your team can easily access the devices they need for testing. xref:organization:teams/search-for-a-team.adoc[Search for a team], then select *Devices* to get started.

== Assign a device

In *Unassiged Devices*, search for a device using filters, statuses, or the search bar.

image:organization:search-devices-closeup.png[width=1000,alt="Search devices in Unassigned devices list"]

Select the *plus* icon to assign the device to your team.

image:organization:click-plus-icon-closeup.png[width=1000,alt="Click plus icon to add a device"]

When you're finished, select *Save*.

image:organization:save-after-assigning-devices-context.png[width=1000,atl="Click Save after assigning devices"]

== Assign all devices

In *Unassigned Devices*, select *Assign All* to assign all devices in your organization to the team.

image:organization:assign-all-closeup.png[width=1000,alt="click Assign all button to add all devices"]

When you're finished, select *Save*.

image:organization:save-assigned-devices-context.png[width=1000,alt="Click Save to save all assigned devices"]

== Automatically assign new public devices

In *Unassigned Devices*, enable *Automatically assign new public device(s) to this team*.

image:organization:add-public-devices-toggle-context.png[width=1000,alt="Toggle to automatically add public devices to the team"]

When you're finished, select *Save*.

image:organization:save-automatically-add-devices-context.png[width=1000,alt="Automatically add devices then Save"]

== Remove a device

In *Assigned Devices*, search for a device using filters, statuses, or the search bar.

image:organization:filter-devices-context.png[width=1000,alt="search for a device using filters, statuses, or the search bar"]

Select the *minus* icon to remove the device from your team.

image:organization:remove-devices-context.png[width=1000,alt="Remove a device from assigned devices list"]

When you're finished, select *Save*.

image:organization:save-remove-devices-context.png[width=1000,alt=""]

== Remove all devices

In *Assigned Devices*, select *Unassign All* to remove all devices currently assigned to your team.

image:organization:unassign-all-context.png[width=1000,alt="Unassign all devices from Assigned devices list"]

When you're finished, select *Save*.
