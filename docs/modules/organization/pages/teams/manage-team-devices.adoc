= Manage team devices
:navtitle: Manage team devices

Learn how to manage your team devices, so your team can easily access the devices they need for testing. xref:teams/view-teams.adoc[View your teams] to get started.

== Assign a device

In *Unassiged Devices*, search for a device using filters, statuses, or the search bar.

image:search-devices-closeup.png[width=1000,alt="Search devices in Unassigned devices list"]

Select the *plus icon* to assign the device to your team.

image:click-plus-icon-closeup.png[width=1000,alt="Click plus icon to add a device"]

When you're finished, select *Save*.

image:save-after-assigning-devices-context.png[width=1000,atl="Click Save after assigning devices"]

== Assign all devices

In *Unassigned Devices*, select *Assign All* to assign all devices in your organization to the team.

image:assign-all-closeup.png[width=1000,alt="click Assign all button to add all devices"]

When you're finished, select *Save*.

image:$NEW-IMAGE$[width=,alt=""]

== Automatically assign new public devices

In *Unassigned Devices*, enable *Automatically assign new public device(s) to this team*.

image:add-public-devices-toggle-context.png[width=1000,alt="Toggle to automatically add public devices to the team"]

When you're finished, select *Save*.

image:$NEW-IMAGE$[width=,alt=""]

== Remove a device

In *Assigned Devices*, search for a device using filters, statuses, or the search bar.

image:filter-devices-context.png[width=1000,alt="search for a device using filters, statuses, or the search bar"]

Select the *minus icon* to remove the device from your team.

image:remove-devices-context.png[width=1000,alt="Remove a device from assigned devices list"]

When you're finished, select *Save*.

image:$NEW-IMAGE$[width=,alt=""]

== Remove all devices

In *Assigned Devices*, select *Unassign All* to remove all devices currently assigned to your team.

image:unassign-all-context.png[width=1000,alt="Unassign all devices from Assigned devices list"]

When you're finished, select *Save*.

image:$NEW-IMAGE$[width=,alt=""]
