= Search for a team
:navtitle: Search for a team

Learn how to search for a team so you can xref:organization:teams/manage-teams.adoc[manage the team], xref:organization:teams/manage-team-devices.adoc[manage their devices], or xref:organization:teams/create-a-device-cleanup-policy.adoc[create a device cleanup policy].

== Search for a team

In Kobiton, select *Org Management*.

image:$NEW-IMAGE$[width="", alt=""]

Select *Teams* to view your teams.

image:select-a-team-context.png[width=1000,alt="Select a Team to get started"]

Use the search bar to filter the list.

image:$NEW-IMAGE$[width="", alt=""]

Select their team name.

image:$NEW-IMAGE$[width="", alt=""]

Now you can xref:organization:teams/manage-teams.adoc[manage the team], xref:organization:teams/manage-team-devices.adoc[manage their devices], or xref:organization:teams/create-a-device-cleanup-policy.adoc[create a device cleanup policy].
