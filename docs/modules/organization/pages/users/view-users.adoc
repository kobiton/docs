= View users
:navtitle: View users

Learn how to view your users so you can invite new users, manage users, and view user history.

== View users

In the Kobiton portal, select *Org Management*.

image:$NEW-IMAGE$[width="", alt=""]

Select *Users* to view your users.

image:invite-users-context.png[width=1000,alt="In Org Management, select Users tab"]

Now you can invite new users, manage users, and view user history.
