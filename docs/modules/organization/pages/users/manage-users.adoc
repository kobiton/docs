= Manage users
:navtitle: Manage users

Learn how to manage your users so you can assign a role, add a user to a team, view user history, and more. xref:users/search-for-a-user.adoc[Choose a user] to get started.

== Enable Single-Sign On (SSO) authentication

After you xref:sso-authentication/about-sso-authentication.adoc[configure SSO authentication for your organization], select *Details*, then *Enable SSO*.

image:enable-sso-closeup.png[width=1000,alt="Select Details, then Enable SSO"]

When you're finished, select *Save*.

image:click-save-enable-sso-context.png[width=1000,alt="Click Save after enabling SSO"]

== Disable a user

In *Details*, select *Disable* to deactivate the user but keep them in your system.

image:disable-user-closeup.png[width=1000,alt="Select Disable to deactivate a user"]

When you're finished, select *Save*.

image:disable-popup-closeup.png[width=500,alt="A closeup to the popup confirming disabling the user"]

== Remove a user

In *Details*, select *Remove* to completely remove the user from your system.

image:remove-users-manage-users-closeup.png[width=1000,alt="A closeup to remove a user"]

When you're finished, select *Save*.

image:remove-confirmation-popup-closeup.png[width=500,alt="A closeup to removing users confirmation popup"]

== Assign a role

In *Roles*, use the search bar to find a role you xref:roles/create-a-role.adoc[previously created].

image:search-for-roles-closeup.png[width=1000,alt="Open Roles tab in Users and search for a role"]

Select the *plus* icon next to the role.

image:add-role-to-user-closeup.png[width=1000,alt="Add a role to a user"]

When you're finished, select *Save*.

image:click-save-add-role-context.png[width=1000,alt="Click Save after adding the role to a user"]

== Remove a role

In *Roles*, select the *minus* icon next to the role you want to remove.

image:remove-role-closeup.png[width=1000,alt="Remove a role from a user"]

When you're finished, select *Save*.

image:click-save-remove-role-context.png[width=1000,alt="Click save after removing a role from a user"]

== Add to a team

In *Teams*, use the search bar to find a team you previously created.

image:teams-tab-context.png[width=1000,alt="In Teams, use the search bar to find a team you previously created."]

Select the *plus* icon next to the team.

image:team-click-plus-icon-closeup.png[width=1000,alt="Select the plus icon next to the team to assign the team for a user"]

When you're finished, select *Save*.

image:click-save-add-teams-context.png[width=1000,alt="Click Save after adding a team to a user"]

== Remove from a team

In *Teams*, select the *minus* icon next to the team you want to remove the user from.

image:remove-from-a-team-closeup.png[width=1000,alt=" select the minus icon next to the team you want to remove the user from"]

When you're finished, select *Save*.

image:click-save-remove-teams-context.png[width=1000,alt="Click Save after removing a team"]
