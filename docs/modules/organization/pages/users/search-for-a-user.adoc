= Search for a user
:navtitle: Search for a user

Learn how to search for a user so you can xref:users/manage-users.adoc[manage the user] or xref:users/search-for-a-user.adoc[view their user history].

== Search for a user

In Kobiton, select *Org Management*.

image:$NEW-IMAGE$[width="", alt=""]

Select *Users* to view all users.

image:invite-users-context.png[width=1000,alt="In Org Management, select Users tab"]

Sort the list using the *up* and *down* arrows, or filter the list using the search bar.

image:$NEW-IMAGE$[width="", alt=""]

Select their username.

image:$NEW-IMAGE$[width="", alt=""]

Now you can xref:users/manage-users.adoc[manage the user] or xref:users/search-for-a-user.adoc[view their user history].
