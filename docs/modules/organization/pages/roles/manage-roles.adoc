= Manage roles
:navtitle: Manage roles

Learn how to manage roles for your organization so you can leverage Role-Based Access Control (RBAC). xref:roles/view-roles.adoc[View your roles] to get started.

[NOTE]
Learn more about xref:about-role-based-access-control.adoc[Role-Based Access Control (RBAC)].

== Rename a role

In *Details*, enter a new name.

image:enter-new-name-closeup.png[width=1000,alt="Enter new name for the role"]

When you're finished, select *Save*.

image:<NEW-IMAGE>[width=,alt=""]

== Change a role description

In *Details*, enter a new description.

image:enter-role-description-context.png[width=1000,alt="Enter new description for the role"]

When you're finished, select *Save*.

image:<NEW-IMAGE>[width=,alt=""]

== Clone a role

In *Details*, select *Clone* to create a copy of the role.

image:clone-role-context.png[width=1000,alt="Clone a role"]

When you're finished, select *Save*.

image:clone-a-role-closeup.png[width=1000,alt="Save after cloning a role"]

== Disable a role

In *Details*, select *Disable* to deactivate the role but keep it in your system.

image:disable-role-context.png[width=1000,alt="Disable a role"]

When you're finished, select *Save*.

image:<NEW-IMAGE>[width=,alt=""]

== Delete a role

In *Details*, select *Delete* to completely remove a role from your system.

image:delete-role-context.png[width=1000,alt="Delete a role"]

When you're finished, select *Save*.

image:confirm-delete-closeup.png[width=500,alt="Confirm deleting a role"]

== Assign a role to a user

In *Members*, select *Add Members*, then search for the user you want to assign the role to.

image:add-member-closeup.png[width=1000,alt="Add members to the role"]

Select the *plus icon* next to the user.

image:<NEW-IMAGE>[width=,alt=""]

When you're finished, select *Save*.

image:assign-a-role-context.png[width=1000,alt="Finish assigning a role to a user"]

== Remove a role from a user

In *Members*, search for the user you want to remove the role from.

image:<NEW-IMAGE>[width=,alt=""]

image::remove-users-closeup.png[width=1000,alt="Remove a user from a role"]

Select the *minus icon* next to the user.

image:<NEW-IMAGE>[width=,alt=""]

When you're finished, select *Save*.

image:<NEW-IMAGE>[width=,alt=""]

image:save-after-remove-context.png[width=1000,alt="Save after removing a user"]

== Set permissions

In *Permissions*, add or remove permissions from the role.

image:<NEW-IMAGE>[width=,alt=""]

When you're finished, select *Save*.

image:assign-remove-permissions-context.png[width=1000,alt="Assign/Remove permissions from the role"]
