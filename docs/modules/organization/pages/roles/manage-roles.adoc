= Manage roles
:navtitle: Manage roles

Learn how to manage roles for your organization so you can leverage Role-Based Access Control (RBAC). xref:organization:roles/search-for-a-role.adoc[Search for a role] to get started.

[NOTE]
Learn more about xref:organization:about-role-based-access-control.adoc[Role-Based Access Control (RBAC)].

== Rename the role

In *Details*, enter a new name.

image:organization:enter-new-name-closeup.png[width=1000,alt="Enter new name for the role"]

When you're finished, select *Save*.

== Change the role description

In *Details*, enter a new description.

image:organization:enter-role-description-context.png[width=1000,alt="Enter new description for the role"]

When you're finished, select *Save*.

== Clone the role

In *Details*, select *Clone* to create a copy of the role.

image:organization:clone-role-context.png[width=1000,alt="Clone a role"]

When you're finished, select *Save*.

image:organization:clone-a-role-closeup.png[width=1000,alt="Save after cloning a role"]

== Disable the role

In *Details*, select *Disable* to deactivate the role but keep it in your system.

image:organization:disable-role-context.png[width=1000,alt="Disable a role"]

When you're finished, select *Save*.

== Delete the role

In *Details*, select *Delete* to completely remove a role from your system.

image:organization:delete-role-context.png[width=1000,alt="Delete a role"]

When you're finished, select *Save*.

image:organization:confirm-delete-closeup.png[width=500,alt="Confirm deleting a role"]

== Assign the role to a user

In *Members*, select *Add Members*, then search for the user you want to assign the role to.

image:organization:add-member-closeup.png[width=1000,alt="Add members to the role"]

Select the *plus* icon next to the user.

image:organization:add-user-to-role-closeup.png[width=1000,alt="Select the plus icon to add a user to a role"]

When you're finished, select *Save*.

image:organization:assign-a-role-context.png[width=1000,alt="Finish assigning a role to a user"]

== Remove the role from a user

In *Members*, search for the user you want to remove the role from.

image:organization:search-remove-user-closeup.png[width=1000,alt="A closeup to search for a user to remove"]

Select the *minus* icon next to the user.

image:organization:remove-users-closeup.png[width=1000,alt="Remove a user from a role"]

When you're finished, select *Save*.

image:organization:save-after-remove-context.png[width=1000,alt="Save after removing a user"]

[#_set_permissions]
== Set permissions

In *Permissions*, add or remove permissions from the role.

[NOTE]
xref:organization:roles/user-roles-and-permissions.adoc[See full list of user roles and permissions].

image:organization:assign-remove-permissions-context.png[width=1000,alt="Assign/Remove permissions from the role"]

When you're finished, select *Save*.
