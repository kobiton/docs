= Search for a role
:navtitle: Search for a role

Learn how to search for a role so you can xref:organization:roles/manage-roles.adoc[manage roles] in your organization.

== Search for a role

In Kobiton, select *Org Management > Roles*.

image:organization:manage-roles-context.png[width=1000,alt="In Org Management, select Roles and choose a role"]

Use the search bar to filter the list of roles.

image:organization:search-bar-role-context.png[width=1000,alt="Use the search bar to filter the roles"]

Select the role's name.

image:organization:select-the-role-context.png[width=1000,alt="Select the role's name"]

Now you can xref:organization:roles/manage-roles.adoc[manage the role].
