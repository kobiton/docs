= Search for a role
:navtitle: Search for a role

Learn how to search for a role so you can xref:roles/manage-roles.adoc[manage a role] in your organization.

== Search for a role

In Kobiton, select *Org Management*.

image:$NEW-IMAGE$[width="", alt=""]

Select *Roles* to view your roles.

image:manage-roles-context.png[width=1000,alt="In Org Management, select Roles and choose a role"]

Use the search bar to filter the list.

image:$NEW-IMAGE$[width="", alt=""]

Select the role's name.

image:$NEW-IMAGE$[width="", alt=""]

Now you can xref:roles/manage-roles.adoc[manage the role].
