= Create a role
:navtitle: Create a role

Learn how to create a role for users in your organization, so you can leverage xref:about-role-based-access-control.adoc[Role-Based Access Control (RBAC)].

== Create a role

In *Org Management*, select *Roles*, then *Create*.

image:create-a-role-context.png[width=1000,alt="Create a role in Org Management"]

In *Details*, enter a role name and description, then select *Members*.

image:name-closeup.png[width=1000,alt="Enter the role's name and description"]

In *Members*, select *Add Members*, then search for the user you want to assign the role to.

image:search-members-closeup.png[width=1000,alt="Search for the user you want to assign the role"]

Select the *plus icon* next to the user, then select *Permissions*.

image:select-plus-icon-closeup.png[width=1000,alt="Select the plus icon next to the user, then select *Permissions"]

In *Permissions*, use the search bar to find permissions.

[NOTE]
xref:roles/user-roles-and-permissions.adoc#_permissions[See full list of user permissions].

image:search-permissions-closeup.png[width=1000,alt="Search for permissions in the search box"]

Select the *plus icon* next to each permission you want to add to the role.

image:add-the-role-closeup.png[width=1000,alt="Add the role to the permission"]

When you're finished, select *Save*.

image:select-save-permission-context.png[width=1000,alt="Select Save to save permission"]
