= View roles
:navtitle: View roles

Learn how to view your roles so you can create and manage roles for your organization.

== View roles

In the Kobiton portal, select *Org Management*.

image:$NEW-IMAGE$[width="", alt=""]

Select *Roles* to view your roles.

image:manage-roles-context.png[width=1000,alt="In Org Management, select Roles and choose a role"]

Now you can create and manage roles for your organization.
