= Search for a device bundles
:navtitle: Search for a device bundles

Learn how to search for a device bundles so you can xref:roles/manage-roles.adoc[manage the device bundle].

== Search for a device bundles

In Kobiton, select *Org Management*.

image:$NEW-IMAGE$[width="", alt=""]

Select *Device Bundles* to view your device bundles.

image:device-bundles-tab-context.png[width=1000,alt="Select Device Bundle in Org Management"]

Select *iOS* or *Android* to filter the list.

image:$NEW-IMAGE$[width="", alt=""]

Select a device bundle.

image:$NEW-IMAGE$[width="", alt=""]

Now you can xref:roles/manage-roles.adoc[manage the device bundle].
