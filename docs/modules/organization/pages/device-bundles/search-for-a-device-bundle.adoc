= Search for a device bundle
:navtitle: Search for a device bundle

Learn how to search for a device bundle so you can xref:organization:roles/manage-roles.adoc[manage the device bundle].

== Search for a device bundle

In Kobiton, select *Org Management*.

image:select-org-management-context.png[width=1000,alt="A context to tab Org Management selected"]

Select *Device Bundles* to view your device bundles.

image:organization:device-bundles-tab-context.png[width=1000,alt="Select Device Bundle in Org Management"]

Select *iOS* or *Android* to filter the list.

image:organization:choose-ios-android-devices-closeup.png[width=1000,alt="Choose iOS or Android to filter the list"]

Select a device bundle.

image:organization:select-a-bundle-context.png[width=1000,alt="A context to a bundle detail"]

Now you can xref:organization:roles/manage-roles.adoc[manage the device bundle].
