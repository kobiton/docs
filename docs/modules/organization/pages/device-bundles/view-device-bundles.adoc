:navtitle: View device bundles

Learn how to view your device bundles so you can create and manage device bundles for your organization.

== View device bundles

In the Kobiton portal, select *Org Management*.

image:$NEW-IMAGE$[width="", alt=""]

Select *Device Bundles* to view your device bundles.

image::device-bundles-tab-context.png[width=1000,alt="Select Device Bundle in Org Management"]

Now you can create and manage device bundles for your organization.
