= Manage device bundles
:navtitle: Manage device bundles

Learn how to manage the device bundles in your organization so users can use them to run xref:scriptless-automation:index.adoc[Scriptless Automation] on devices in parallel. You can only manage device bundles xref:device-bundles/create-a-device-bundle.adoc[you or your team created].

== View a device bundle

In Kobiton, select *Org Management*, then select *Device Bundles*.

image:select-device-bundles-context.png[width=1000,alt="Select Device Bundles tab"]

xref:organization:device-bundles/search-for-a-device-bundle.adoc[Search for a device bundle], then select the *Bundle Name*.

== Manage the device bundle

=== Rename the bundle

Select the *Edit* icon next to a device bundle.

image:organization:edit-bundle-closeup.png[width=1000,alt="Click pencil to edit the device bundle"]

Enter a new name for the bundle, then select *Save Bundle*.

image:organization:click-save-bundle-after-edit.png[width=1000,alt="Click Save Bundle after editing"]

=== Add or remove devices

Select the *Edit* icon next to a device bundle.

image:organization:edit-bundle-closeup.png[width=1000,alt="Click pencil to edit the device bundle"]

Manually search, or use the search bar to find devices.

image:organization:apply-remove-devices-closeup.png[width=500,alt="Manually search, or use the search bar to find devices"]

Select *Apply* to add or remove devices, then select *Save Bundle*.

image:organization:click-save-bundles-manage-devices-closeup.png[width=500,alt="Select *Apply* to add or remove devices, then select *Save Bundle*"]

=== Delete the bundle

Select the *Remove* icon next to a device bundle.

image:organization:delete-bundle-closeup.png[width=1000,alt="A closeup to delete function of Device Bundle"]

Select *Yes* to confirm.

image:organization:yes-no-modal-closeup.png[width=500,alt="Click Yes to confirm delete the bundle"]
