= Manage device bundles
:navtitle: Manage device bundles

Learn how to manage the device bundles in your organization so users can use them to run xref:scriptless-automation:index.adoc[Scriptless automation] on devices in parallel. xref:device-bundles/view-device-bundles.adoc[View your device bundles] to get started.

== Rename bundle

Select the *Edit* icon next to a device bundle.

image::edit-bundle-closeup.png[width=1000,alt="Click pencil to edit the device bundle"]

Enter a new name for the bundle, then select *Save Bundle*.

image::click-save-bundle-after-edit.png[width=1000,alt="Click Save Bundle after editing"]

== Add or remove devices

Select the *Edit* icon next to a device bundle.

image:edit-bundle-closeup.png[width=1000,alt="Click pencil to edit the device bundle"]

Manually search, or use the search bar to find devices.

image:apply-remove-devices-closeup.png[width=500,alt="Manually search, or use the search bar to find devices"]

Select *Apply* to add or remove devices, then select *Save Bundle*.

image:click-save-bundles-manage-devices-closeup.png[width=500,alt="Select *Apply* to add or remove devices, then select *Save Bundle*"]

== Delete a bundle

Select the *Remove* icon next to a device bundle.

image:delete-bundle-closeup.png[width=1000,alt="A closeup to delete function of Device Bundle"]

Select *Yes* to confirm.

image:yes-no-modal-closeup.png[width=500,alt="Click Yes to confirm delete the bundle"]
