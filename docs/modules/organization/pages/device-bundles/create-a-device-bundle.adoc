= Create a device bundle
:navtitle: Create a device bundle

Learn how to create a new device bundle so users in your organization can run xref:scriptless-automation:index.adoc[Scriptless Automation] on devices in parallel.

[NOTE]
Device bundles are operating system (OS) specific so Android and iOS devices cannot be bundled together.

== Create a new bundle

In *Org Management*, select *Device Bundles*.

image:organization:device-bundles-tab-context.png[width=1000,alt="Select Device Bundle tab in Org Management"]

Choose *iOS* or *Android*, then select *New Bundle*.

image:organization:choose-os-closeup.png[width=1000,alt="Choose iOS or Android, then select New Bundle"]

Enter a name for your bundle.

image:organization:type-bundle-name-closeup.png[width=400,alt="Type New Bundle name field"]

Manually search, or use the search bar to find devices.

image:organization:search-device-closeup.png[width=500,alt="Manually search, or use the search bar to find devices"]

Select *Apply* to add a device to the bundle.

image:organization:apply-devices-closeup.png[width=500,alt="Select Apply to add devices to a bundle"]

When you're finished, select *Save Bundle*.

image:organization:save-bundle-context.png[width=1000,alt="click Save Bundle when already applied devices"]
