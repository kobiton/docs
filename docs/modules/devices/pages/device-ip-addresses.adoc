= Device IP addresses
:navtitle: Device IP addresses

These are the IP address for Kobiton devices. You can use a device IP address to xref:local-devices/connect-a-device-to-your-private-network.adoc[connect devices to your private network].

== Public and private devices

You can use the following IP addresses for any public or private device:

* `173.213.234.130`
* `68.64.33.132`
* `64.19.222.254`

== Proof of Concept (POC)

You can use the following IP address for POC devices:

* `173.213.234.130`
