= Session Explorer timeline
:navtitle: Timeline

The Session Explorer timeline lets you play back sessions, share test steps, view accessibility issues, and more.

image:timeline-context.png[width=1000,alt="A context to Timeline in Session Explorer"]

== Play or pause recording

Play or pause the session recording.

image:play_pause-closeup.png[width=400,alt="A closeup to play/pause button in the timeline"]

== Test steps

Display the selected test step on the device screen.

image:test-steps-closeup.png[width=1000,alt="A closeup to test steps displayed on the timeline"]

The related action is marked in red.

image:red-dot-context.png[width=1000,alt="The related action is marked in red."]

== Previous or next test step

Skip to the previous or next test step.

image:previous-next-closeup.png[width=1000,alt="A closeup to Previous & Next button on the timeline"]

Zoom in or out of the session timeline.

image:zoom.png[width=1000,alt="A closeup to Zoom function on the timeline"]

== Accessibility issue marker

Session Explorer automatically flags accessibility issues during the session. Select *View Details* to open xref:session-explorer/accessability-validation.adoc[accessibility validation] and view all issues in one place.

image:issue-marker-context.png[width=1000,alt="Session Explorer automatically flags accessbility issues."]
