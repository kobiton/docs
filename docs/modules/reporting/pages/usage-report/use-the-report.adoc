= Use the usage report
:navtitle: Use the report

Learn how to use the usage report so you can view component usage across all user test sessions. See xref:reporting:usage-report/report-metadata.adoc[report metadata] to learn more about each field and data type in the report.

== Open the report

In Kobiton, select *Reports*.

image:$NEW-IMAGE$[width=,alt=""]

Select *Usage Report* to open the report.

image:reporting:usage-report-context.png[width=1000,alt="A context of Usage report in Predefined report"]

== Filter the report

[#_time_range]
=== By time period

To filter the report by a predefined time period, select the *Period* dropdown.

image:reporting:time-range-closeup.png[width=500,alt="A closeup to time range that affects columns of Usage report"]

Choose *Last 7 days*, *Last 15 days*, or *Last 30 days*, then select the *Refresh* icon.

image:$NEW-IMAGE$[width=,alt=""]

=== By custom date range

To filter the report by a custom date range, select the *Period* dropdown, the choose *Custom*.

image:$NEW-IMAGE$[width=,alt=""]

Choose any range up to 90 days, then select the *Refresh* icon.

image:$NEW-IMAGE$[width=,alt=""]
