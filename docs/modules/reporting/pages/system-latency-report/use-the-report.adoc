= Use the system latency report
:navtitle: Use the report

Learn how to use the system latency report so you can view how long it takes the system to perform an action. For more information, see xref:reporting:system-latency-report/report-metadata.adoc[].

== Open the report

In Kobiton, select *Analytics Reports*.

== Review the data

The report presents data across two tabs: *By Connection* and *By Location*.

=== By Connection

The By Connection tab contains the following tables:

* Connection Summary
* Connection Location Summary
* Connection deviceShare Summary
* Connection Details
