= Use the device availability report
:navtitle: Use the report

Learn how to use the device availability report so you can view availability data for your devices. For more information, see xref:reporting:device-availability-report/report-metadata.adoc[].

== Open the report

In Kobiton, select *Reports*.

image:reporting:select-report-context.png[width=1000,alt="A context to the Reports feature selected"]

Select *Device Availability Report* to open the report.

image:reporting:device-availability-report-context.png[width=1000,alt="Device Availability report in Predefined report"]

== Use the report

=== Get the latest data

To get the latest data for your report, select *Refresh*.

image:reporting:date-range-closeup.png[width=500,alt="A date range in Device Availability report"]

=== Filter the report

[#_by_date]
==== By date

To filter the report to show outages within a specific date range, choose any range up to 90 days, then select *Refresh*. This range is applied to each xref:reporting:device-availability-report/report-metadata.adoc#_report_types[report type] and xref:reporting:device-availability-report/report-metadata.adoc#_report_fields[field].

image:reporting:date-range-closeup.png[width=500,alt="A date range in Device Availability report"]

==== By device UDID

To filter the report by one or more device UDIDs, enter a UDID into the search bar, then check the UDIDs you'd like to use.

image:reporting:filter-by-udid-context.png[width=1000,alt="Filter devices by UDID"]

=== Add a note

To add a note for a specific outage, select the *pencil* icon.

image:reporting:select-pencils-closeup.png[width=1000,alt="Select the pencil icon"]

To exclude this outage from your SLA, check the box.

image:reporting:exclude-sla-closeup.png[width=500,alt="Check the exclude from SLA Calculation"]

When you're finished, select *Save*.
