= Manage apps
:navtitle: Manage apps

Learn how to manage apps in the app repository so you and your team can install apps xref:devices:install-an-app.adoc[before] or xref:manual-testing:install-an-app.adoc[during] a test session.

== Open the app repository

In Kobiton, select *Apps* to open the app repository.

image:apps:select-apps-context.png[width=1000,alt="Select Apps feature on Kobiton Portal"]

[#_upload_an_app]
== Upload an app

To upload an app using the Kobiton portal, select the *+* icon and choose a xref:apps:supported-filetypes.adoc[supported filetype] from your computer.

image:apps:add-app-context.png[width=1000,alt="A context to the plus icon to upload the app"]

[NOTE]
You can also upload an app xref:upload-an-app/using-the-kobiton-api.adoc[using our REST API].

[#_rename_an_app]
== Rename an app

To rename an app, select *Rename Application* and enter a new name.

image:apps:rename-an-app-context.png[width=1000,alt="A context to Renmae an app"]

[#_remove_an_app]
== Remove an app

To remove an app from your repository, select *Remove App*.

image:apps:remove-apps-closeup.png[width=300,alt=" A closeup to Remove App"]

To remove a specific version instead, select the app first, then select *Remove App* next to the version.

image:apps:remove-a-version-of-the-app-closeup.png[width=1000,alt="A closeup to remove a version of the app"]

== Download an app version

To download an app to your computer, select *Download*.

image:apps:download-apps-closeup.png[width=300,alt="A closeup to Download the current app"]

To download a specific version instead, select the app first, then select *Download* next to the version.

image:apps:download-another-version-of-the-app-closeup.png[width=1000,alt="A closeup to Download a version of the app"]

== Limit access

To limit who can access the app, select the *globe* icon and choose *Allow access to my team(s)* or *Allow only me to access*.

image:apps:app-privacy-closeup.png[width=300,alt="A closeup to access Limit access of the app"]

[NOTE]
See your teams in xref:organization:teams/search-for-a-team.adoc[organization management].
