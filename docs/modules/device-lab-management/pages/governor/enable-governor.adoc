= Enable governor for parallel tests
:navtitle: Enable governor

Learn how to enable and configure the Max Parallel Tests for a hosting machine.

== Prerequisites

* A Kobiton account with *ADMIN* role.
* Private or local devices.

== Enable governor

Open the Kobiton web portal and log in using an account with *ADMIN* role.

Select the profile picture and choose **Settings**, then choose **Device Management**.

Choose *Settings* under a host machine.

In the pop-up, choose either of the below options:

* *Unlimited* (default): no limit for the host machine.

* *Set custom limit*: set a custom limit from 1 to 999.

Select *Save*.
