= Device lab management
:navtitle: Device lab management

In this section, you will learn about what tools and services Kobiton has and how you can use and manage them. You will learn about:

- Hardware requirement of deviceConnect, which necessitates an M1 or M2 Mac mini.
- Restarting deviceConnect services to apply configuration changes or software updates.
- Retrieving deviceConnect logs for troubleshooting and submitting tickets to Kobiton Support.
